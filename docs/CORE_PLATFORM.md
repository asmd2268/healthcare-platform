# أساس المنصة المشترك

تم تنفيذ أساس Next.js/TypeScript داخل `apps/web` فقط، مع App Router وServer Components افتراضيًا. يوفر النواة: RTL/LTR، العربية الافتراضية والإنجليزية الكاملة، theme، Shell متجاوب، إعدادات علامة مركزية، عقود للهوية والصلاحيات والتدقيق وسير العمل والإشعارات والاستيراد/التصدير، وطبقة Supabase مفصولة للمتصفح والخادم.

المصادقة، التراجع، الإشعارات الفعلية، الملفات، قاعدة البيانات، والتراخيص ما زالت Contracts أو واجهات Placeholder واضحة وليست وظائف إنتاج. لا توجد وحدات صحية أو بيانات مرضى أو ترحيل Legacy في هذا التغيير.
