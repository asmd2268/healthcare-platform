# الصلاحيات

## القاعدة

تفويض الوصول يتم على الخادم لكل منشأة ووحدة وسجل وإجراء. الدور يحدد الحد الأقصى، وقد تمنح صلاحيات إضافية محددة المدة مع سجل موافقة. لا يرى المستخدم بيانات منشأة أخرى أو ملاحظات سرية دون منح صريح.

## مصفوفة الصلاحيات

الأفعال القياسية: عرض، إنشاء، تعديل مسودة، إرسال، تعليق عام، تعليق سري، تعيين، اعتماد، إغلاق، إعادة فتح، تصدير، إدارة قوالب، وإدارة إعدادات. يملك مسؤول المنشأة الضبط، ويملك المعتمد قرار الاعتماد دون أن يعتمد عمله، ويملك المدقق وصولًا للقراءة والتدقيق حسب التفويض.

## ضوابط خاصة

التصدير، عرض المرفقات الحساسة، وإدارة المستخدمين تحتاج صلاحيات منفصلة. تراجع الصلاحيات دوريًا وعند تغيير الوظيفة أو انتهاء الوصول. تسجل منح الصلاحية وسحبها واستخدام الإجراءات عالية الأثر.
