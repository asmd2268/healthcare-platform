# إرشادات UI/UX

تصمم الواجهة بالعربية أولًا وRTL افتراضيًا، مع إنجليزية LTR كاملة. يستخدم خط Tajawal/ثمانية العربي بعد التحقق من الترخيص، وأحجام واضحة ومسافات متسقة. يدعم كل سطح الوضع النهاري والليلي، التباين، لوحة المفاتيح، قارئات الشاشة، والنصوص البديلة.

تتبع النماذج: عنوان واضح، حقول لازمة مميزة، تحقق قريب من الحقل، حفظ تلقائي مع حالة مرئية، ومسودة قابلة للاستئناف. تعتمد القوائم بحثًا، فرزًا، فلاتر، نطاق تاريخ، تصديرًا حسب الصلاحية، وحالة فارغة ذات إجراء. لا تستخدم الألوان وحدها للدلالة على الخطر أو الحالة.

تظهر الإجراءات الحساسة (اعتماد، إغلاق، إعادة فتح، حذف منطقي، تصدير) تأكيدًا يشرح الأثر ويطلب سببًا عند الحاجة. الملاحظات السرية تميّز بصريًا ولا تظهر في الطباعة أو التصدير إلا للمصرح لهم.
