# محرك لوحات المعلومات

Dashboard Definition وDashboard Widget مستقلان عن الوحدة. الأنواع المدعومة: KPI Card، Counter، Gauge، Line، Area، Bar، Stacked Bar، Pie، Donut، Heatmap، Trend، Table، Top N، Timeline، وActivity Feed. كل widget يتصل بإصدار تقرير منشور وعقد configuration allowlisted؛ لا يوجد تنفيذ script في المتصفح.

لوحات Medication Errors وDepartment Inspections في الواجهة تعريفات تجريبية لا تستبدل لوحات الوحدات الحالية ولا تعرض بيانات حقيقية. عند تشغيل adapters ستغطي مؤشرات أخطاء الدواء المطلوبة والتفتيش من مصادر مصرح بها.
