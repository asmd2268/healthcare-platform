# إدارة المنصة

أساس مشترك لمالك المنصة ومدير المنظمة ومدير المنشأة، بعقود server-side-compatible لنطاقات global/organization/facility والصلاحيات الإدارية. لا يتجاوز `platform.full_access` نطاق tenant أو organization أو facility؛ مالك المنصة فقط مع role `platform_owner` ونطاق global يصل لكل المستأجرين. شاشات `/administration/*` Placeholder فقط؛ لا توجد قاعدة بيانات أو تنفيذ خلفي أو SQL خام أو enforcement خلفي فعلي بعد.
