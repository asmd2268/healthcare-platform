# التراخيص

تدعم المنصة تراخيص شهرية وسنوية ودائمة، وEnterprise وTrial، ومحددة بالمنشأة أو المستخدم أو المرفق والوحدات والحدود المتفق عليها. تتحقق الخدمة الخلفية من الترخيص دون تعطيل وصول الطوارئ أو تعريض البيانات للفقد؛ تحدد سياسة فترة السماح والوصول للقراءة بعد الانتهاء تعاقديًا.

White Label وترخيص الوحدات المستقلة وخيارات cloud وOn-premises قابلة للضبط. يدعم الترخيص تاريخ الانتهاء، وفترة السماح، والاستحقاقات، وتنشيط الوحدة والمستأجر، والتحقق، والسيناريوهات غير المتصلة، والسجل التدقيقي. يبقى منطق الترخيص وقيمه الحساسة على الخادم. توثق الفواتير والتجديد والتعليق والنقل والدعم والحقوق التجارية، وتراجع تراخيص الخطوط والمكتبات والمكونات قبل اعتمادها.
