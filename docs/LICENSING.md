# التراخيص

تدعم المنصة تراخيص شهرية وسنوية ودائمة، محددة بالمنشأة والوحدات وعدد المستخدمين أو الحدود المتفق عليها. تتحقق الخدمة الخلفية من الترخيص دون تعطيل وصول الطوارئ أو تعريض البيانات للفقد؛ تحدد سياسة فترة السماح والوصول للقراءة بعد الانتهاء تعاقديًا.

White Label وترخيص الوحدات المستقلة خيارات قابلة للضبط. توثق الفواتير والتجديد والتعليق والنقل والدعم والحقوق التجارية، وتراجع تراخيص الخطوط والمكتبات والمكونات قبل اعتمادها.
