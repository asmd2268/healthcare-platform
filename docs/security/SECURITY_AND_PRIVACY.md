# Security and privacy baseline

## Status

المتطلبات الأمنية الأساسية ملزمة كما في `PROJECT_BIBLE.md` و`docs/SECURITY.md`. تبقى المتطلبات القانونية والتنظيمية التفصيلية حسب الاختصاص القضائي والمنشأة قيد التأكيد؛ ولا يوقف ذلك ضوابط الأمن الأساسية.

## Baseline expectations

- Classify all data before collection or processing begins.
- Apply least-privilege access controls and preserve auditable access records.
- Protect sensitive data in transit and at rest using approved controls.
- Keep secrets out of source control and manage them through an approved secret-management process.
- Define retention, deletion, incident response, and vendor-review processes before handling regulated data.
- Validate applicable privacy, health-data, and security obligations with the organization’s legal and compliance owners.

This document is a planning baseline, not legal or compliance advice.
