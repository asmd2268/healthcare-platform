# واجهة إدارة المنصة

توفر الواجهة الحالية Dashboard ومكتبة نماذج وBuilder وReference Data وArchived Records وDeletion Requests وImport/Export Centers عبر Repository تجريبي. البيانات لا تحفظ في قاعدة بيانات؛ Builder محلي وعمليات الحذف والاستيراد والتصدير الخلفية Placeholder فقط. يتطلب الإنتاج مصادقة حقيقية وتفويض server-side وSupabase persistence قبل أي إجراء إداري.
