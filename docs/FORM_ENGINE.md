# محرك النماذج الشامل

يدعم العقود مسودات، إصدارات، حماية النشر، حقول عربية/إنجليزية، منطقًا شرطيًا وتحقيقًا وخرائط Word/Excel/CSV/JSON/Google Sheets. يدعم المخطط الآن `form_definitions` و`form_versions` و`form_sections` و`form_fields`، مع ترتيب وrevision وحقول ثنائية اللغة. يمنع Trigger تعديل أو حذف النسخة المنشورة، وتظل السجلات التاريخية قادرة على حفظ exact form version.

لا تنفذ imports/exports الفعلية أو publication workflow المعتمد بالكامل بعد؛ هذه واجهات وjobs مستقبلية تتطلب صلاحية وتدقيقًا.
