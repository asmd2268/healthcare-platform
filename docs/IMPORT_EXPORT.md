# الاستيراد والتصدير

يدعم CSV وExcel وJSON وPDF حسب نوع السجل والغرض. الاستيراد يمر باختيار قالب وإصدار، تعيين أعمدة، تحقق من الصيغة والقيم والصلاحيات، معاينة صفوف وأخطاء، ثم تأكيد قابل للتدقيق. لا يستورد العميل مباشرة إلى جداول الإنتاج بلا تحقق خلفي.

التصدير يراعي المنشأة والدور والفلاتر المطبقة؛ يوضح الحقول المستبعدة والوقت ومن نفذه، وقد يحتاج سببًا أو موافقة للبيانات الحساسة. تضاف علامة المنشأة ومعلومات الإصدار إلى PDF عند ملاءمة ذلك. تحفظ ملفات التصدير مؤقتًا وفق سياسة احتفاظ ولا تتضمن أسرارًا.
