# الاستيراد والتصدير

يدعم التصدير CSV وExcel وJSON وPDF عند الملاءمة، والاستيراد CSV وExcel وJSON. يمر الاستيراد باختيار قالب وإصدار، وتحديد نمط التشغيل (Append أو Merge أو Update existing records أو Full replacement)، وتعيين أعمدة، وتحقيق الصيغة والقيم والصلاحيات، ومعاينة صفوف وأخطاء، ثم تأكيد قابل للتدقيق. لا يستورد العميل مباشرة إلى جداول الإنتاج بلا تحقق خلفي.

يتطلب Full replacement تأكيدًا صريحًا، ونسخة احتياطية، والتحقق من الصلاحية، والتحقق الكامل للاستيراد، وتقرير أخطاء، وإدخالًا في Audit Log. لا يستخدم لتجاوز سياسة الاحتفاظ أو الاعتماد أو عزل المستأجر.

التصدير يراعي المنشأة والدور والفلاتر المطبقة؛ يوضح الحقول المستبعدة والوقت ومن نفذه، وقد يحتاج سببًا أو موافقة للبيانات الحساسة. تضاف علامة المنشأة ومعلومات الإصدار إلى PDF عند ملاءمة ذلك. تحفظ ملفات التصدير مؤقتًا وفق سياسة احتفاظ ولا تتضمن أسرارًا.
