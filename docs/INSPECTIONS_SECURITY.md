# أمن التفتيشات

العقود الحالية لا تحفظ بيانات أو ملفات. عند ربط التخزين، يطبق server-side tenant/organization/permission checks، ولا يعامل إخفاء الزر كصلاحية. رفع الصور والمرفقات Placeholder حتى فحص النوع والحجم والبرمجيات الخبيثة والتخزين الخاص والروابط الموقعة. لا يوجد CAPA أو Audit persistence فعلي؛ تسجل العقود الأحداث المستقبلية فقط.
