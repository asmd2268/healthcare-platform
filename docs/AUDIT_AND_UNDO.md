# التدقيق والتراجع الآمن

كل تغيير مؤثر يسجل: السجل والوحدة والمنشأة، الفاعل، الوقت، الإجراء، الحالة السابقة واللاحقة أو ملخص فرق آمن، ومصدر العملية. سجل التدقيق ليس بديلًا عن سجل النشاط التشغيلي، ولا يعدله المستخدم.

تتيح المنصة سجل تغييرات آخر 24 ساعة للمستخدمين المصرح لهم. التراجع يعكس تغييرًا محددًا ضمن الصلاحية فقط، وينشئ حدث تدقيق جديدًا؛ لا يمحو الاعتمادات أو الصادرات أو الأدلة، ولا يتجاوز تعديلات لاحقة أو تعارضات. بعد 24 ساعة يستخدم مسار إعادة فتح/تصحيح رسمي مع سبب وموافقة عند اللزوم.
