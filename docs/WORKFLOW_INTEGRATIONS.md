# تكاملات Workflow

يوفر Medication Errors adapter mapping محدودًا للحالات الموجودة فقط ولا يزيل safeguards قاعدة بياناته. Inspection adapter عقد مستقبلي لأحداث started/draft/submitted/under_review/approved/findings_follow_up/closed/reopened؛ لا يعيد كتابة وحدة التفتيش. تدعم الوحدات الأخرى adapters مستقلة كي تبقى قابلة للبيع والنشر المنفصل.
