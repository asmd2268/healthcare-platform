# الإجراءات التصحيحية والوقائية (CAPA)

يستقبل CAPA مصدرًا مرتبطًا ببلاغ أو تفتيش أو استدعاء أو إنشاء يدوي، ويحدد المشكلة والسبب والخطة والمالك والمواعيد ومقياس الفعالية. يدعم مهامًا فرعية وأدلة وتعليقات وتصعيدات وتأخيرات مبررة. الحالات: مسودة، مفتوح، قيد التنفيذ، بانتظار تحقق الفعالية، بانتظار اعتماد، مغلق، أو معاد فتحه. لا يُغلق قبل التحقق المستقل من الفعالية وتوثيق القرار.
