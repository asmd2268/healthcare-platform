# تقارير أخطاء الدواء

تعرض هذه الوحدة مؤشرات البلاغات المفتوحة والمتأخرة، الشدة، تكرار التصنيف، الاتجاهات، زمن الاستجابة، اكتمال RCA، وفعالية CAPA. تسمح بفلاتر المنشأة والقسم والفترة ونوع الحدث، ومقارنة فترات متكافئة دون كشف بيانات لا يملكها المستخدم. تدعم PDF للعرض وCSV/Excel/JSON للتحليل حسب الصلاحية، مع تعريفات موثقة لكل مقياس وتنبيه عند صغر العينة أو نقص البيانات.
