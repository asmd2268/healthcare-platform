# اللوحات والتقارير

تقدم لوحات بحسب الدور والمنشأة: عناصر مفتوحة ومتأخرة، نتائج التفتيش، CAPA، أخطاء الدواء، الاستدعاءات، والاتجاهات. كل مقياس يملك تعريفًا ومصدرًا وفلاتر فترة وقسم ووحدة، مع انتقال آمن للتفاصيل. تدعم التقارير المقارنة والتنزيل المصرح به، وتحترم الصلاحيات والملاحظات السرية. لا تعرض مؤشرات مضللة عند نقص البيانات أو اختلاف القوالب دون تنبيه.

كل تقرير يتيح Arabic أو English أو Bilingual. يشمل اختيار اللغة عنوان التقرير وأسماء الحقول والجداول والرسوم البيانية والتواريخ والرأس والتذييل وشعار المنشأة وتوقيعات الاعتماد وعبارة «تطوير: علي أبودهش / Developed by Ali Abudahash». يحترم الإخراج RTL/LTR وتنسيق الأرقام والتاريخ حسب الاختيار، ولا يترجم تلقائيًا حقول المحتوى الحر أو السرية. تختبر اللوحات والفلاتر والتسميات والتحميل والتصدير باللغتين قبل قبولها.
