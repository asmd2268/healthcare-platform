# اللوحات والتقارير

تقدم لوحات بحسب الدور والمنشأة: عناصر مفتوحة ومتأخرة، نتائج التفتيش، CAPA، أخطاء الدواء، الاستدعاءات، والاتجاهات. كل مقياس يملك تعريفًا ومصدرًا وفلاتر فترة وقسم ووحدة، مع انتقال آمن للتفاصيل. تدعم التقارير المقارنة والتنزيل المصرح به، وتحترم الصلاحيات والملاحظات السرية. لا تعرض مؤشرات مضللة عند نقص البيانات أو اختلاف القوالب دون تنبيه.
