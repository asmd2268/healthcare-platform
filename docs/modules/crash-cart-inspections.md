# تفتيش عربات الطوارئ (Crash Cart Inspections)

توفر قوائم فحص مجدولة حسب الموقع والعربة، مع عناصر الأجهزة والأدوية والختم والانتهاء والنظافة والجاهزية. يسجل المفتش نتيجة، قيمة عند الحاجة، صورة، وملاحظة؛ ويخلق مخالفة أو مهمة عند عدم المطابقة. تدعم توقيع المفتش والمراجع، التنبيهات للتفتيش المتأخر أو الفشل الحرج، وسجلًا لا يُستبدل للفحوصات السابقة.
