# استدعاءات الأدوية

تدير إشعار الاستدعاء، مصدره، مستوى الخطورة، المنتج والدفعة، المناطق المتأثرة، وخطة العزل والاسترجاع. تنشئ مهام للأقسام وتتابع الكميات المحددة والمعزولة والمسترجعة، الإشعارات، الأدلة، والتصعيد. لا يُغلق الاستدعاء قبل مراجعة الاكتمال وتوثيق الاستثناءات والاعتماد؛ وتنتج الوحدة تقرير امتثال زمني.
