# السياسات والإجراءات (Policies & Procedures)

ترفع الوحدة ملفات السياسات مع المالك والتصنيف والجمهور والإصدار وحالة المسودة. تدير دورة المراجعة والتعليقات والاقتراحات، الاعتماد، النشر، تاريخ المراجعة والانتهاء، والتذكيرات. يحتفظ النظام بكل الإصدارات السابقة غير القابلة للاستبدال، ويعرض النسخة المنشورة فقط للجمهور المخول.

## اللغة والترجمة

لا يفترض النظام أن السياسات عربية؛ الافتراضي المقترح للسياسات الحالية هو الإنجليزية مع إمكان إضافة ترجمة عربية لاحقًا. تدعم الوحدة سياسة English فقط، أو Arabic فقط، أو Bilingual داخل ملف واحد، أو نسختين منفصلتين مرتبطتين بوصفهما ترجمتين للسياسة ذاتها. لكل سياسة وإصدار تُحفظ: **Document Language** (Arabic/English/Bilingual)، **Original Language**، **Translation Status**، **Translation Reviewer**، **Translation Approval Status**، **Linked Translation Version**، و**Translation Last Updated Date**.

لا تنشر ترجمة مرتبطة بوصفها مكافئة للنسخة الأصلية قبل اعتماد الترجمة. يحتفظ الرابط بين النسختين بالسياسة والإصدار لا بالملف فقط، كي يتضح عند تغيير الأصل أن الترجمة تحتاج مراجعة. يمكن أن تكون بيانات الاسم والوصف والكلمات المفتاحية عربية أو إنجليزية أو ثنائية حسب الحاجة، ولا يجبر مالك السياسة على إنشاء نسخة ثانية.

يمكن للموظف الإقرار بالقراءة مع الوقت والإصدار واللغة المعروضة، وتتابع الإدارة الناقصين وتذكرهم. يوفر البحث السريع بالعناوين والكلمات المفتاحية العربية والإنجليزية، ورقم السياسة، ومحتوى الوثيقة عند توفر البحث النصي، مع قابلية إضافة المرادفات في اللغتين لاحقًا. يملك كل إصدار منشور QR Code يصل للنسخة المصرح بها. لا تغير التعليقات أو الاقتراحات الملف المعتمد؛ تتحول إلى مراجعة أو إصدار جديد بعد قرار المالك.
