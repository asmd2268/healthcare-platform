# مخزون الأقسام (Floorstock)

توثق الوحدة أصناف مخزون الأقسام، الحد الأدنى والأقصى، الدفعات والانتهاء، الجرد، الطلب، الاستلام، التسوية، والهدر. لا تعتمد التسوية أو الصرف الاستثنائي دون صلاحية وسبب. توفر مقارنة الاستهلاك والانحراف والنقص حسب القسم والفترة، وتدعم الاستيراد المراجع والتصدير المصرح به.
