# عربة الصيدلية (Pharmacy Trolley)

تدير الوحدة تعريف العربات، مواقعها، المسؤول عنها، الأصناف والكميات المستهدفة، تواريخ الانتهاء، وحالة الجاهزية. تسجل الجولات والتعبئة والنقل والفقد والملاحظات مع صور عند الحاجة. تنبه للنقص والانتهاء وتنتج قائمة عمل للفني أو الصيدلي، مع تاريخ كامل للحركة وإقفال بعد المراجعة.
