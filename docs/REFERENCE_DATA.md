# البيانات المرجعية

تشمل العقود facilities وbranches وdepartments وjob titles وmedication categories وerror types وseverity levels وinspection types وfinding statuses وpolicy categories وdocument types وrecall classifications وequipment types وقيم dropdown مخصصة.

المخطط يضيف `reference_data_groups` و`reference_data_items` مع labels عربية/إنجليزية وcode وترتيب وactive/archive ونطاق tenant/organization/facility وقيود uniqueness وrevision. الأرشفة هي السلوك المقصود؛ الحذف الدائم للقيم المرجعية المرتبطة بالسجل التاريخي غير منفذ.
